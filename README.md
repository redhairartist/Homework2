Задание 2
string - это строка, котрую нужно обрезать и повторить. offset = 0 обозначает количество символов, которые нужно обрезать слева (по умолчанию = 0). repetitions = 1 это количество повторений строки (по умолчанию 1). Команда string[offset:] — возвращает часть строки, начиная с символа с индексом offset и до конца строки. trimmed_string * repetitions — повторяет строку trimmed_string указанное количество раз (repetitions)
